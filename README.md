## Build Instruction
Add the following files to common:
```
project_root/
|-- common/
    |-- includes/
        |-- glm/
        |-- ShaderProgram.h
        |-- stb_image.h
    |-- src/
        |-- ShaderProgram.cpp
```
Move shaders/ and relevant image texture files to the build directory
